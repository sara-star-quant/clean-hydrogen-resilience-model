# Code of Conduct

This project adopts the Contributor Covenant version 2.1 as its code of conduct.

The full text and the reporting guide are published at the canonical source:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (issues, pull requests, discussions, the code, the
documentation, the playground), you agree to follow that document.

## Reporting

Concerns about behaviour in this project should be sent privately to the maintainer
listed in the GitHub profile, or via the email address recorded there. Reports are
handled per the Contributor Covenant guide linked above.
